# React Webpack, SCSS & Typescript Boilerplate

Fun coding quizzes for non-programmers.

## Getting Started

1. Install dependencies

```
cd boilerplate-react-webpack && npm install
```

2. Run development server

```
npm start
```

3. Build package

```
npm run build
```

4. Lint code

```
npm run lint
```

4. Format code

```
npm run format
```
